How do I search images?

You can search images by heading over to the homepage or the posts section. You can search for a term in a simple manner using the Simple Search or use the Advanced Search to filter posts down to exactly what you need.

Online Service and Local Generation

By default, the website lets you upload and search for images in a way that fits locally generated images with a lot of options, like the ones generated in Stable Diffusion's Web UI by AUTOMATIC1111.

While this works great, some online services like Midjourney are a lot simpler, and fit everything in their prompts. For this reason, those services can be searched by checking the online service checkbox.

Search fields

  • Prompt / Search Term:
    • This parameter searches prompts and is generally what you want to use to search for what you have in mind (e.g: dog, car, river).
  • Model:
    • This parameter lets you search for a specific model. It is only used when searching for locally generated images likes ones generated in the Stable Diffusion Web UI.
  • Keywords:
    • Keywords search metadata fields like notes and generation details. This is useful to look for images that use a particular sampling method or any other attribute or word that would show up in those fields.
  • Service:
    • Lets you choose between many different online services that are used to generated AI art.
